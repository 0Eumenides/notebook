# 场景结构

一个threejs程序结构由以下几点构成：

  • 场景Scene
    • 网络模型Mesh
      • 几何体对象Geometry
      • 材质对象Material
    • 光照Light（平行光，环境光，点光源...）
  • 相机Camera
  • 渲染器Renderer
  • 图片

# 场景对象

  1. 创建方式

    var scene = new THREE.Scene();

  2. 建立几何体对象Geometry

    //创建一个球体几何对象
var geometry = new THREE.SphereGeometry(60, 40, 40);
//创建一个立方体几何对象
var geometry = new THREE.BoxGeometry(10, 100, 100);

  3. 建立材质对象Material

    var material = new THREE.MeshLambertMaterial({
 color: 0xff0000
});

  4. 将Geometry和Material合并为Mesh网格模型

    var mesh = new THREE.Mesh(geometry, material);

  5. 将网格模型插入到场景对象中

    scene.add(mesh);

  6. 添加光照

    光照的种类有很多种，如平行光，环境光，点光源等等

    设置点光源：

    var point = new THREE.PointLight(0xffffff);
point.position.set(400, 200, 300); //点光源位置

    设置环境光

    var ambient = new THREE.AmbientLight(0x444444);

    将光源加入到场景中

    scene.add(point);
scene.add(ambient);

# 相机

控制显示的场景范围，以后的代码调试会用到

// 相机设置
var width = window.innerWidth;  //窗口宽度
var height = window.innerHeight;  //窗口高度
var k = width / height    //窗口宽高比
var s = 100;  //三维场景显示范围控制系数，系数远大，显示的范围越大
var camera = new THREE.OrthographicCamera(-s * k, s * k,
  s, -s, 1, 1000); //创建相机对象
camera.position.set(200, 300, 200); //设置相机位置
camera.lookAt(scene.position);  //设置相机方向（指向场景对象）

# 渲染器

场景、相机、渲染器是三个相互独立JavaScript对象

  1. 创建渲染器

    var renderer = new THREE.WebGLRenderer();
renderer.setSize(width, height);  //设置渲染区域尺寸
renderer.setClearColor(0xb9d3ff, 1);  //设置背景颜色
document.body.appendChild(renderer.domElement); //body元素中插入canvas对象

  2. 场景和相机对象作为渲染器.render方法的参数出现，执行渲染操作，指定场景，相机作为参数

    renderer.render(scene, camera);

    render方法可以简单的理解为是对WebGL API绘制函数drawArrays()的封装

    执行一次render方法，产生的渲染结果会覆盖上次执行render方法的渲染结果

# 完整代码

// 创建场景对象Scene
var scene = new THREE.Scene();


//创建网格模型
//创建一个球体几何对象
var geometry = new THREE.SphereGeometry(60, 40, 40);
//创建一个立方体几何对象
// var geometry = new THREE.BoxGeometry(10, 100, 100);

var material = new THREE.MeshLambertMaterial({
  color: 0xff0000
});//材质对象Material
var mesh = new THREE.Mesh(geometry, material);//网格模型对象Mesh
scene.add(mesh); //网格模型添加到场景中


//光源设置
//点光源
var point = new THREE.PointLight(0xffffff);
point.position.set(400, 200, 300);    //点光源位置
scene.add(point);   //点光源添加到场景中
//环境光
var ambient = new THREE.AmbientLight(0x444444);
scene.add(ambient);


// 相机设置
var width = window.innerWidth;  //窗口宽度
var height = window.innerHeight;  //窗口高度
var k = width / height    //窗口宽高比
var s = 100;  //三维场景显示范围控制系数，系数远大，显示的范围越大
var camera = new THREE.OrthographicCamera(-s * k, s * k,
  s, -s, 1, 1000); //创建相机对象
camera.position.set(200, 300, 200); //设置相机位置
camera.lookAt(scene.position);  //设置相机方向（指向场景对象）


//创建渲染器对象
var renderer = new THREE.WebGLRenderer();
renderer.setSize(width, height);  //设置渲染区域尺寸
renderer.setClearColor(0xb9d3ff, 1);  //设置背景颜色
document.body.appendChild(renderer.domElement); //body元素中插入canvas对象
//执行渲染操作，指定场景，相机作为参数
renderer.render(scene, camera);

# requestAnimationFrame

我们想到得到动画效果，需要对模型进行重复渲染

可以使用setInterval(函数,时间间隔)函数，它会按照一定的周期执行函数

而window.requestAnimationFrame方法，它会请求浏览器执行下一帧的动画。

在刷新频率方面，不用指定时间间隔，浏览器会自动安排，一般最高刷新频率默认60FPS，如果两帧之间需要大量的计算，刷新频率也可能会低于60FPS，如果太低的话会出现卡顿现象。

不同的threejs应用场景，执行一帧的时间是不确定的，threejs动画程序一般选择requestAnimationFrame方法。

//渲染函数
function render() {
  renderer.render(scene, camera);
  mesh.rotateY(0.01);
  requestAnimationFrame(render);
}
render();

# 鼠标控制

先导入OrbitControls.js文件，文件在threejs项目的examples\js\controls目录下

<script src="lib/OrbitControls.js"></script>

//创建控件对象，控件可以监听鼠标的变化，改变相机对象的属性
var controls = new THREE.OrbitControls(camera, renderer.domElement);
//监听鼠标事件，触发渲染函数，更新canvas画布渲染效果
controls.addEventListener('change', render);
